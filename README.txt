To play the game, download the TapSoccer_Data folder and TapSoccer.exe
Most of the code is in Assets/Scripts/
The resolution is buggy but setting it to 1280x600 seems to work

For 1 button controls:
Left players jump: Z
Right players jump: X

For 2 button controls:
Left players jump: Z and X
Right players jump: left arrow and right arrow key
